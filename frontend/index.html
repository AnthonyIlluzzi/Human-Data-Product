Human Data Product

Modeling my professional career as a real data product.

This project explores a simple idea: What if my professional experience was structured, queryable, and consumable like a modern data platform?

Instead of presenting experience as a static resume, this system models career history as a structured dataset with APIs, analytical views, and insight generation.

It demonstrates how a human-centered dataset can behave like a real data product.

Data Product Definition

A data product is a curated, reusable, and governed data asset often derived from fragmented data sources. They are pre-packaged 'products' that include not just the data itself, but also access paths, semantical context, metadata, transformations, and documentation. They are designed to enable better decision making, and therefore must be reliable, discoverable, and provide analytical usability.
