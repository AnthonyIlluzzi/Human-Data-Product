# Human Data Product

This project treats a professional resume as a structured data product.

Instead of a static document, the dataset can be:

• queried via SQL  
• consumed through an API  
• explored through an insights layer  

The goal is to demonstrate how data products enable information to become accessible, explorable, and useful.

## Architecture

Sources → Transformation → Structured Data Model → Consumption Interfaces

Interfaces include:

- REST API
- SQL query interface
- Insight explorer
